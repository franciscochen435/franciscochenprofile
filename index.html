Campus Navigation AI Agent Using Reinforcement Learning

Modeled a campus-scale navigation problem as a high-dimensional Markov Decision Process (88K+ discrete states), incorporating stochastic weather conditions, crowd density, and hierarchical routing (surface vs. underground tunnel).
Designed and implemented a modular, OpenAI Gym-compatible simulation environment supporting configurable transition dynamics and reward components.
Developed an ε-greedy Q-learning agent with adaptive exploration decay to balance exploration–exploitation trade-offs.
Trained the agent over 1M episodes and monitored convergence via cumulative reward trends and policy stability analysis.
Benchmarked performance against shortest-path and random-walk baselines, achieving ~6.5-step average goal completion with emergent weather-aware tunnel selection behavior.
